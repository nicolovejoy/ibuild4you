**Shipped 2026-06-15 PM (#70 welcome-replay + cost-runaway fix; Phase-0 sweep PR open):** **#70 (PR #76, `672303f`):** stop replaying the static `welcome_message` as message-zero on every new session. `app/api/sessions/route.ts` inserts the canned welcome only on the project's **first** session; on return sessions the kickoff recaps instead. `lib/agent/kickoff.ts` + the kickoff route now judge "prior maker history" at the **project** level (`last_maker_message_at`), so a blank builder-pre-created return session still earns a state-aware recap; true first-ever sessions still decline. `MakerProjectView` passes project history into the predicate + suppresses the type-to-create race. **Playwright-verified on preview** (created session 2 as owner → 0 canned messages; opened as maker → Sam recapped "Hey Mara, welcome back…"). **🔥 Cost-runaway fix (PR #78, `f1b8177`):** diagnosed via `api_usage` logs — one prod brief ("NWMLS Contract Generator") called `brief.generate` **229×/day** at exactly the 5-min cron interval (~$8.4/day). Root cause: that brief always fails regen (`max_tokens`, payload > `BRIEF_MAX_TOKENS`), and the circuit breaker cleared the counter on a new maker message but kept the **stale** `failures_since`, so it cleared-and-retried (billing Sonnet) forever. Fix is three-layered: **(A)** extracted the streak math into a pure tested module `lib/api/brief-regen-gate.ts` — a maker message newer than the streak resets the WHOLE streak and a fresh failure re-anchors `failures_since` to now, so it trips after 3 and HOLDS; **(B)** `BRIEF_MAX_TOKENS` 2048→8192 so a large brief regenerates once and goes quiet; **(C)** cron skips briefs every member has archived. Emergency stop `scripts/stop-regen-loop.mjs` halted the live loop. 689 tests green. **Phase-0 sweep PR #77 (open, on preview):** #66 admin-menu was already shipped (verified); #67 reminder-status strip on Setup (new `nextReminderAt()` helper); #68 tolerant JSON import (`parseLooseJson` — try-parse then structural repair of smart-quote delimiters/NBSP; valid content never rewritten).
